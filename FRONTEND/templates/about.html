About EEG Classification Project

This project aims to classify neurological disorders using EEG signals. We employ advanced machine learning and deep learning models, including Random Forest, XGBoost, and neural networks, to analyze EEG data and predict disorders such as Anxiety, Mood Disorders, Schizophrenia, and more.

Our system allows users to:

  • Upload EEG datasets for analysis.
  • View model accuracies and performance metrics.
  • Obtain predictions for new EEG data samples.

The models were trained on the EEG dataset, achieving high accuracy through feature selection, preprocessing, and hyperparameter tuning. This web application provides an intuitive interface for researchers and clinicians to interact with the models.
